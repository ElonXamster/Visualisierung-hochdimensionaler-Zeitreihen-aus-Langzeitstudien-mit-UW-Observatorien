The generation of high-dimensional data from the LoVe Ocean Observatory
and an interest in understanding and visualizing this data, created the need for a tool to
accomplish this.

The high-dimensional data was reduced to manageable dimensions by three different, renowned algorithms (PCA, t-SNE, UMAP).
The flexible way to select the data for these algorithms was a priority to allow for different analyses of the data, which gives the opportunity to explore different perspectives.

The visualization of high-dimensional data from the LoVe Ocean Observatory, using Python and its libraries, was successfully done by reducing the
dimensions and displaying the low-dimensional representation in the gui of the ”LoVe Visualization tool”.
